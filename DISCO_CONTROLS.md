# 🕺 DISCO Controls

The alternate Analogue Pocket button map for **Heretic** and **Hexen**,
selected from `Options > MORE… > CONTROLS` (the default is the unified
layout in [CONTROLS.md](CONTROLS.md)). DISCO keeps the extra actions on
**B chords** and the shoulders free for strafe / jump. Doom has no DISCO
variant — it uses the one map in [CONTROLS.md](CONTROLS.md).

## 🐉 Heretic

**B is a modifier**: tap it to use/open, or hold it and press another
button for the chorded action below.

| Button | In game | In menus |
| --- | --- | --- |
| D-pad | Move / turn | Navigate |
| A | Fire | Confirm |
| B (tap) | Use / open | Back |
| X | Use inventory item | — |
| Y | Next weapon | — |
| L1 / R1 | Strafe left / right | — |
| Start | Menu | Close menu |
| Select | Automap | — |

**Hold B + …**

| Chord | Action |
| --- | --- |
| B + Up / Down | Fly up / down |
| B + Left / Right | Select previous / next inventory item |
| B + Y | Previous weapon |

## 🛡️ Hexen

Same idea as Heretic. Hexen adds jumping on the shoulders, so strafe
moves onto a B chord.

| Button | In game | In menus |

| A | Fire | Confirm |
| B (tap) | Use / open | Back |
| X | Use inventory item | — |
| Y | Next weapon | — |
| L / R | Jump | — |
| Start | Menu | Close menu |
| Select | Automap | — |

**Hold B + …**

| Chord | Action |
| --- | --- |
| B + Up / Down | Fly up / down |
| B + Left / Right | Strafe left / right |
| B + Y | Previous weapon |
| B + L / R | Select previous / next inventory item |
